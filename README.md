# Network Security Architecture Analysis

## Overview
This project evaluates network protection strategies for a retail enterprise environment with over 1,000 users. The focus is on securing transaction systems, protecting inventory data, and implementing scalable security solutions with limited IT resources.

## Objectives
- Protect transaction integrity
- Monitor and control inventory access
- Detect and prevent unauthorized access
- Minimize infrastructure costs while maintaining security

## Key Concepts
- Defense in Depth (Layered Security)
- Least Privilege
- Complete Mediation
- Confidentiality & Integrity (CIA Triad)

## Proposed Solution
A hybrid approach combining:
- Network Intrusion Detection System (NIDS)
- Host Intrusion Prevention System (HIPS)

## Why This Matters
This project demonstrates how security decisions must balance:
- Risk
- Cost
- Scalability
- Operational constraints
